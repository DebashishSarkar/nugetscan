# nugetscan
Consolidated report on nuget packages for a project/solution

### Report Content
- ID
- Resolved Version
- Latest Version
- Status: Vulnerable | Outdated | Deprecated | Up-To-Date
- Notes

### Report Format
- JSON
- CSV

### Underlying approach
- Use dotnet list command options to get nuget info
- Consolidate into a unified report
